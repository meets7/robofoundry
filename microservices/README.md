
# Endpoints:

* Read only: 
   * List of domains, packages and robots: 
        ```http://[hostname]/robofoundry/api/read/robots ```
   * Code of single robot:
   
     ``` http://[hostname]/robofoundry/api/read/robots/?        robotid=crazy&userid=User&packageid=sample```
